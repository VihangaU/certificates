# 🎓 Certificate Portfolio

A modern, responsive web application to showcase professional certifications and achievements. Built with vanilla HTML, CSS, and JavaScript, featuring a beautiful dark theme with gradient accents and smooth animations.

## ✨ Features

### 🎨 Modern Design
- **Dark Theme**: Professional dark color scheme with gradient accents
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Entrance animations, hover effects, and transitions
- **Gradient Text**: Eye-catching gradient titles matching portfolio style

### 🔍 Interactive Features
- **Category Filtering**: Filter certificates by technology/domain
  - Web Development
  - Mobile Development
  - Cloud & Backend
  - Security
  - Architecture
- **Modal Viewer**: Click any certificate to view in fullscreen
- **Hover Effects**: Interactive card hover animations
- **Auto-hiding Navigation**: Navigation bar hides on scroll down, appears on scroll up

### 📱 Responsive Design
- **Desktop Navigation**: Full navigation bar for large screens
- **Mobile Menu**: Hamburger menu with slide-out sidebar for mobile devices
- **Touch Optimized**: Optimized for touch interactions on mobile

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern CSS with CSS Variables, Flexbox, and Grid
- **JavaScript (ES6+)**: Vanilla JavaScript for interactivity
- **Google Fonts**: Poppins font family

## 🎨 Design Philosophy

This project follows modern web design principles:
- **Mobile-First**: Designed for mobile, enhanced for desktop
- **Performance**: Optimized for fast load times
- **Accessibility**: Keyboard navigation and semantic HTML
- **User Experience**: Smooth animations and clear visual hierarchy
- **Consistency**: Matches portfolio website design language
